MyPizza

WWW

Contexte:

En autonomie, nous devions créer un site web de pizzeria. Le but de ce projet était de créer un petit projet, afin de maîtriser les bases en PHP, javascript. La gestion de la base de donnée s'est faite avec MySQL. À la fin j'ai décidé de changer le stack technique pour optimiser mon travail et mon choix s'est porté sur le Framework PHP: Laravel.

Missions:

  • Permettre au client de commander des articles en ligne
  • Créer une page d'administration, pour gérer les commandes et les produits.
  • Concevoir la fonctionnalité: Notifications
  • Appel à des apis externes
    • API Pusher: Traitement des données en temps réel
    • API Gouv adresse: Vérification des adresses en fonction de la ville choisie
  • Mise en place d'une Base de donnée en MYSQL
  • Développement du projet avec le framework php Laravel
